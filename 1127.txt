Broadcast address(virtual address)
- 시스템 운영, 프로토콜 운영, 애플리케이션 운영 시
- 운영체제 설치 시 할당 
- BIP : 255.255.255.255 (all)
         192.168.1.255
  BMAC : FFFF.FFFF.FFFF

Multicast address(virtual address)
- 시스템 운영, 프로토콜 운영, 애플리케이션 운영 시
- 운영체제 설치 시 할당, 애플리케이션 설치 시
- MIP : 224~239

운영체제 설치 시 
 2개  브로드캐스트 주소
      255.255.255.255 /FFFF.FFFF.FFFF
      192.168.1.255 / FFFF.FFFF.FFFF
게이트웨이 
 - 망과 망을 연결시켜 주는 중계 장비 
 - 라우터, 멀티레이어 스위치, 게이트웨이(방화벽)

허브( 1계층)
 - 경로데이타베이스 : 없음
 - 전송 방법 :  플러딩 

스위치(2계층)
 - 경로데이타베이스 : Mac address table
 - 전송 방법
   * 포워딩 : (수신지 MAC 주소)가 (MAT )에 있는 경우 
   * 플러딩 : (수신지 MAC 주소 )가 (MAT )에 없는 경우
- 브로드캐스트/멀티캐스트 패킷에 대해서는 플러딩 방식으로 전송

라우터( 3계층)
 - 경로데이타베이스 : Routing table
 - 전송 방법
   * 포워딩 : (수신지 IP 주소)가 (RT )에 있는 경우 
   * 폐기(drop) : (수신지 IP 주소)가 (RT )에  없는 경우
- 브로드캐스트/멀티캐스트 패킷에 대해서는 폐기(전송하지 않음) 

*  경로 데이타베이스에는 브로드캐스트/멀티캐스트 주소를 등록안함
